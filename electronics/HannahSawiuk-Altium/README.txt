Component Libraries:
A collection of libraries downloaded from SnapEDA. 
Some not used.

Schematic_Library:
Contains "chicker.SCHLIB" which is a library of the components used in the chicker schematic. 
This file offers easier access to the components and their footprints.

IMPORTANT
Do not remove any of the libraries.
"chicker.SCHLIB" will be useless without the original project libraries.
